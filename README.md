# Aplicación de Finanzas Personales con IA Predictiva
Aplicación web de finanzas personales que facilita el control de ingresos y gastos mediante una interfaz intuitiva. 
El sistema integra un módulo de inteligencia artificial predictiva que analiza los hábitos financieros del usuario y genera recomendaciones personalizadas, alertas y proyecciones de gasto, contribuyendo a una mejor gestión financiera personal.
